Install

url https://www.rust-lang.org/zh-CN/tools/install

while you input enter ，will install rust default

上面这条命令会安装一些内容，比如 cargo rustup
如果你想要使用他们，需要将他们添加到环境变量中

update

更新版本, rustup用来管理rust的版本,rust每6个礼拜会进行一次版本迭代，使用rustup update更新版本

cargo工具

Cargo：Rust 的构建工具和包管理器

cargo 使用

  • Cargo.toml中包含了编译你项目时所需要的内容清单
  • src/main.rs 中包含了主程序

使用cargo build 构建项目,编译完成之后，会在生成一个编译后的可执行文件，执行他们

使用cargo run 构建项目并运行。

具体是使用文档
https://doc.rust-lang.org/cargo/guide/index.html
